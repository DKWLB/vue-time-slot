# Vue Time Slot

A tiny timeslot selector component with no dependencies for your app!

[Demo]()

## Installation

`npm install vue-time-slot --save`

### Usage

```javascript
import VueTimeSlot from "vue-time-slot";

new Vue({
  el: "#app",

  components: {
    VueTimeSlot
  }
});
```

```html
<vue-time-slot :min="5" ></vue-loop>
```

## Optional Props

| Prop |  Type  | Default |
| ---- | :----: | :-----: |
| Min  | Number |    5    |

## Features

* Lightweight
* No dependencies
